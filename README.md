# Árvore de Metas — CNJ

Organograma interativo de desdobramento de metas.

## Hierarquia

Empresa → Regional → Cluster → Loja → Categoria → Espécie

## Funcionalidades

- Meta por período com visualização personalizada
- Distribuição linear proporcional ao intervalo selecionado
- Pesos por faturamento real (lojas e clusters)
- Quantidade de vendas por ticket médio por categoria
- Export CSV completo
- Painel de configuração de percentuais

## Arquivo principal

`organograma_metas.html` — abre direto no navegador, sem dependências.
